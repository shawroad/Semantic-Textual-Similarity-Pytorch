## 评价指标的说明
- **皮尔逊系数(pearsonr)**: 是衡量两个连续型变量的线性相关关系。 
- **斯皮尔曼相关系数(spearmanr)**: 是衡量两变量之间的单调关系，两个变量同时变化，但是并非同样速率变化，即并非一定是线性关系。

## 实验结果: 
没有专门去调参。 无监督的模型从训练集中随机采样了10000条数据。下面是在测试集上的结果。对最终结果影响比较大的就是学习率。尽可能的小就行。
